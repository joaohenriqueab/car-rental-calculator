# Car Rental Calculator 

Car rental cost calculator in Python with full input validation.

### Description
A simple Python program that calculates the total cost of renting a car based on the number of days rented and miles driven.

### Pricing
- $10.00 per day
- $0.10 per mile driven

### Features
- Robust input validation
- Rejects negative values and zero days
- Error handling with `try/except` (doesn't crash on letters/symbols)
- Loops until valid data is entered
- Clean formatted output

### Example Output

Put the miles traveled here: 100
Put the days rent here: 3
The total value in dollars will be: US$40.00
